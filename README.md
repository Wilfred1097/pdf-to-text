**This project is all about extracting the questions, options, and the correct answer from the questionnaire that is in PDF format. and exporting it into a json format.**

###### Features:

> 1, The system displays the total number of questions detected in the uploaded PDF, along with the number of unanswered questions.
> 2, If the system is unable to recognize the correct answer for a specific question, you can manually assign it by selecting the appropriate option from the dropdown menu.
> 3, You may export the processed data as a JSON file for your own use or integration into other systems.

_The content of your pdf file should be in this format_
```html
1, Which of the following is commonly added to concrete slabs to improve thermal insulation? 
 A) Fiberglass mesh
 B) Expanding agents
 C) Insulating materials like Styrofoam or polystyrene
 D) Colored dyes
 Answer: C
2. What is the typical lifespan of a well-maintained concrete slab under normal conditions?
 A) 5 to 10 years
 B) 10 to 30 years 
 C)30 to 50 years
 D) Over 100 years 
 Answer: C
```
You can try it [here.](https://wilfred1097.github.io/pdf-to-text/)
